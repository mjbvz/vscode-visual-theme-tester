# Tab Demo Project

This project demonstrates different types of tabs in VS Code:

## Features
- TypeScript components
- CSS styling
- HTML templates
- Python utilities
- JSON configuration

## Usage
Run `npm start` to begin development.

## File Types
- `.ts` - TypeScript source files
- `.css` - Stylesheets
- `.html` - HTML templates
- `.py` - Python scripts
- `.json` - Configuration files
